# Chihaya UI - Next.js Sandbox

This is a Next.js sandbox app for testing @chihaya-ui/react components.

## Getting Started

Install dependencies:

```bash
bun install
```

Run the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Usage

This sandbox demonstrates the usage of Chihaya UI components in a Next.js environment.
